# Scrubbing for Jackpots
An idle cliker game where players washing dishes to fund their scratch-off lottery and achieve financial freedom

### Core Gameplay Loop
1. Wash dishes
2. Earn wage
3. Buy scratch tickets
4. Upgrade equipment

### Minimum Viable Product
- [ ] A clickable dirty plate.
- [ ] A Shop
- [ ] A basic GUI for game.
- [ ] A time management system.
- [ ] A money counter.
- [ ] A basic shop to buy lottery.
- [ ] A random number generator to determine if the ticket wins.
